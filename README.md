# EasyMeme

**BNB Chain 生态的智能 Meme 币猎手** — 一款面向高频交易者的 AI 驱动型代币发现与自动化交易工具。

本工具专为寻求 BNB 链极速获利的交易者设计，核心逻辑是通过 AI 实时推演与自动化执行，化繁为简，捕捉"金狗"。

## 核心功能模块

### 模块一：智能扫描与预警系统

实时监控 BNB Chain 上 PancakeSwap、Four.meme 等 DEX 的新池创建事件。

| 子功能 | 详细说明 |
|--------|----------|
| **新池监控** | 监听 PairCreated 事件，<500ms 延迟推送 |
| **多维度过滤** | 流动性阈值、持币分布、创建者历史、社交热度 |
| **风险标签** | 🟢安全 🟡警告 🔴高危（基于 AI 评分） |
| **推送通道** | App 内通知 / Telegram Bot / Discord Webhook |

**监控数据指标：**
- 初始流动性（LP）金额
- 持币地址数 & 前10持仓占比
- 创建者地址历史行为
- 社交媒体提及热度（Twitter/Telegram）

### 模块二：AI 智能合约推演引擎

对目标代币合约进行静态分析 + 动态模拟，识别潜在风险。

| 检测维度 | 检测内容 |
|----------|----------|
| **权限风险** | Owner 权限是否过大、是否可 mint、是否可 pause |
| **交易限制** | 买卖税率、是否有黑名单、冷却期限制 |
| **貔貅检测** | 模拟卖出交易，验证是否可正常售出 |
| **流动性锁定** | LP 是否锁定、锁定时长、锁定平台可信度 |
| **代码相似度** | 与已知 Rug 合约的代码相似度匹配 |

**输出报告示例：**
```
┌─────────────────────────────────────┐
│ Token: $EXAMPLE                     │
│ 风险评分: 72/100 (中等风险)          │
├─────────────────────────────────────┤
│ ✅ LP 已锁定 180 天 (PinkLock)       │
│ ✅ 无 mint 权限                      │
│ ⚠️ 卖出税率 5% (高于平均)            │
│ ❌ Owner 可修改税率 (存在后门风险)    │
└─────────────────────────────────────┘
```

### 模块三：一键交易与跟单系统

#### 手动快捷买入
- 预设金额档位：0.1 / 0.5 / 1 / 5 BNB
- 滑点自动优化（基于当前盘口深度）
- Gas 加速选项（Normal / Fast / Instant）
- MEV 保护选项（通过私有交易池）

#### 自动跟单策略
| 策略类型 | 触发条件 | 配置项 |
|----------|----------|--------|
| **地址跟单** | 监控地址买入时自动跟 | 跟单地址列表、单笔上限、日限额 |
| **热度跟单** | Twitter 提及量激增 | 热度阈值、关键词过滤 |
| **大单跟单** | 检测到大额买单 | 金额阈值、反应延迟 |
| **新池狙击** | 新池创建且通过安全检测 | 最低 LP、最高 MC |

### 模块四：智能止盈止损系统

| 功能 | 说明 |
|------|------|
| **阶梯止盈** | 2x 卖 30%，5x 卖 30%，10x 卖剩余 |
| **追踪止损** | 从最高点回撤 20% 自动卖出 |
| **智能拆单** | 根据盘口深度自动拆分大单，减少滑点 |
| **紧急撤退** | 检测到 Owner 异常操作时自动清仓 |

### 技术选型

| 组件 | 技术选型 | 理由 |
|------|----------|------|
| **链上监听** | Go + WebSocket | 高并发、低延迟 |
| **合约分析** | Python + Slither + LLM | 成熟的审计工具链 |
| **交易执行** | Node.js + ethers.js | 生态成熟、SDK 完善 |
| **前端** | Next.js 14 + Tailwind + RainbowKit | SSR + Web3 钱包集成 |
| **消息推送** | Telegram Bot API + FCM | 覆盖主流渠道 |

## 风险与合规

### 技术风险
| 风险 | 缓解措施 |
|------|----------|
| RPC 节点故障 | 多节点冗余 + 自建节点 |
| 交易执行失败 | 重试机制 + Gas 动态调整 |
| MEV 攻击 | 私有交易池 + Flashbots 集成 |

### 合规风险
| 风险 | 缓解措施 |
|------|----------|
| 资金托管风险 | **非托管架构**，私钥始终在用户端 |
| 投资建议风险 | 明确声明"仅供参考，不构成投资建议" |
| 地区合规 | 地理围栏 + 服务条款明确管辖权 |

### 用户资金安全
- 私钥/助记词永不上传服务器
- 交易签名全部在客户端完成
- 支持硬件钱包（Ledger）
- 可选的交易限额保护
